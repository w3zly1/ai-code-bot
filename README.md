# 🤖 AI Code Assistant v7 ULTRA MONSTER

Мощный Telegram-бот на Python с Gemini AI для помощи с кодом.

## 🔥 Возможности

### Основные
- 💻 Генерация полного рабочего кода
- 🐛 Дебаг и исправление ошибок
- 📖 Объяснение кода простым языком
- ✨ Рефакторинг и улучшение кода
- 🔍 Code review
- 🏗 Проектирование архитектуры

### Продвинутые
- 🚀 Создание проектов командой AI-агентов (6 специализированных агентов)
- 📦 Генерация ZIP-архива с проектом
- 📁 Анализ загруженных файлов
- 🧠 Умное определение режима работы
- 💾 Память диалога с автосжатием
- 🎯 Извлечение технологий из контекста

### Инструменты
- 🔍 Поиск через DuckDuckGo
- 💰 Курсы криптовалют
- 📊 Статистика использования

### Система ролей
- 👤 User — базовый лимит
- ⭐ Premium — расширенный лимит
- 👑 Admin — без лимитов + управление
